About the experiment

This experiment uses angularJS binding property to fetch data automatically from a remote server using public web api of best buy. Because the property of textbox is linked to angularJS model element, whatever changes we make in the UI (textbox) also reflects in the model. The model property of search box is linked to the search url as query parameter. Hence api call is issued on every keyevent made on the search box, thereby dynamically updating the query parameter.

Code Snippet:

$scope.fetchProducts = function () {
var itemType = $scope.title;
$http.jsonp('http://api.remix.bestbuy.com/v1/products(manufacturer=' + itemType +
')?format=json&show=all&callback=JSON_CALLBACK&apiKey=wu48f36djjruzarbeahrx8ay')
.success(function (resp) {
$scope.products = resp.products;
console.log(resp.products);
$(".prdtTable").show();
})
.error(function (data) {
$scope.data = "Request failed";
});

This type of search will help users by showing the results even before the user could finish typing thereby saving their time and also showing suggestions.
