About the experiment

This experiment creates animated buttons by rotating them through 360deg and displaying a different text when mouse hovers them. The buttons are rotated creating an animation effect using transform property. The degree by which the button has to be rotated must be mentioned in the transform property.

Code Snippet:

@keyframes rotate {
0% {
transform: rotate(0deg);
}

100% {
transform: rotate(-360deg);
}
}


The time taken to rotate the button is specified through transition property. Time is mentioned in seconds. Different transition properties has been mentioned for different browsers as the syntax is different for various browsers.

Code Snippet:

.round {
display: block;
position: absolute;
text-shadow: 0 1px 0 rgba(255,255,255,.7);
-webkit-transition: all .25s ease-in-out;
-o-transition: all .25s ease-in-out;
-moz-transition: all .25s ease-in-out;
transition: all .25s ease-in-out;
box-shadow: 2px 2px 7px rgba(0,0,0,.2);
border-radius: 200px;
}
