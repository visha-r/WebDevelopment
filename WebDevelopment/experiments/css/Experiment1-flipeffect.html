CSS Experiment 1 - Flipping animation

Flipping animation

Hover the mouse over the image to see the flipping effect!!

About the experiment

Flipping effect helps to bring out some animation effect like by flipping contents like images on some event . The image above can be flipped by hovering the mouse over it. The effect can also be triggered by different events like onclick, onfocus, onmouseleave etc. The effect works by laying two images(front & back) one over the other at absolute position and only one is visible at a time while the other is hidden.

Code Snippet:

.front, .back {
backface-visibility: hidden;
position: absolute;
top:0px;
left:0px;
}


The images are flipped in a 3-dimensional manner through transform-style property. Also the time it takes to flip between images is controlled through transition property. In this example, the transition time is given as 1 second.

Code Snippet:

.flipper {
transition: 1s;
transform-style: preserve-3d;
}


Because there are two images laid one over the other, the decision to display which image on the top is done through transform property. The following code snippet says the image at the back is rotated 180deg along Y so it will look hidden at the back of front image.

Code Snippet:

.back {
transform: rotateY(180deg);
}


Finally, this flipping effect must be triggered on some event. In this example it is done on mousehover event. When the mouse hovers on the image, it is rotated through 180deg to show the other image.

Code Snippet:

.flip-container:hover .flipper {
transform: rotateY(180deg);
}
