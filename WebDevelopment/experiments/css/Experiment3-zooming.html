CSS Experiment 3 - Zooming & Rotation

Zooming & Rotation

About the experiment

Zooming can bring out some animation effect by enlarging the focused content. The image or the content which is going to be enlarged must be given a size. So when the same image is enlarged, we can mention to what proportion of the original size the image has to be enlarged.

Code Snippet:

.expt{
width:150px;
height:150px;
}

.thumbnail:hover .expt {
transform: scale(1.5) rotate(100deg);
-webkit-transition-timing-function:ease;
}

The styling above says that, when mouse hovers over the image it has be enlarged 1.5 times more than its original size. The image has also been rotated through transform property. The image will be rotated by the given angle bringing out the rotation effect.
