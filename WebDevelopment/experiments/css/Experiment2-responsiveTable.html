About the experiment

Responsive web design (RWD) is an approach to web design aimed at crafting sites to provide an optimal viewing experience—easy reading and navigation with a minimum of resizing, panning, and scrolling—across a wide range of devices. This experiment aims at rendering responsive design of the table and other contents in the page. Responsive design through CSS is done with the following code snippet:

Code Snippet:

@media screen and (min-width:600px) {
.responsiveTable tr:nth-child(odd) {
background-color: lightyellow;
}
.responsiveTable tr:nth-child(even) {
background-color: lightcyan;
}
table th {
background-color: green;
}
table th, td {
padding: 20px 20px 20px;
text-align: center;
}
}


@media screen and (min-width:600px) describes how the page contents should be rendered if the screen' minimum width is 600px. The styles listed inside this tag will style the elements accordingly under the given size. In this case, the table's odd rows will look lightyellow while the even rows will be in lightcyan.

Code Snippet:

.main-wrapper{
width:600px;
margin:auto;
}
.miniTable{
display:none;
}


Because the container width should be different for different screen size, it is controlled through width element. Also, to display different table design at different screen size, 'display' element is used to hide and show the table.
