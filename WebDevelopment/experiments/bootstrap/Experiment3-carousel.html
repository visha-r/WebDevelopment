About the experiment

Carousel :

Bottstrap has in-built support for carousel plugin, which helps to display images in a carousel manner i.e like a slide show. Carousel requires an id for the whole container so functions can be applied to each element in the container.
The class="carousel slide" specifies that this is going to display images in a slide show. The dots at the bottom of each image is called indicator which indicates the position of the current image. The class="left carousel-control" & class="right carousel-control" are used to control the slide show.

Code Snippet:

<a class="right carousel-control" href="#myCarousel" role="button" data-slide="next">
<span class="glyphicon glyphicon-chevron-right" aria-hidden="true"></span>
<span class="sr-only">Next</span>
</a>

To add caption to the pictures, the class "carousel-caption" has to be used.

Code Snippet:

<div class="carousel-caption">
<h3>Dark</h3>
<p>A dark modern wooden background.</p>
</div>

JS Collapse :

JS collapse can be used to collapse and display the menu or navigation items. The support of JS collapse is available through a bootstrap plugin. It also requires jQuery library to support bootstrap collapse. The links or items that have to collapsed must use the classes "panel-collapse" & "collapse".

Code Snippet:

<div id="bootstrap" class="panel-collapse collapse">
<div class="panel-body">
<ul>
<li><a href="../bootstrap/Experiment1-bootstrapElements.html" target="_blank">Bootstrap Glyphicons,Progressbar,grid styling,badge & form-groups</a></li>
<li><a href="../bootstrap/Experiment2-table_with_pagination.html" target="_blank"> Bootstrap Pagination,table and menu </a></li>
</ul>
</div>
</div>

The heading that should appear after collapsing uses the class "panel-heading" and the actual content uses "panel-body".
