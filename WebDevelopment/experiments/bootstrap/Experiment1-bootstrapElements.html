About the experiment

Bootstrap is a collection of pre-defined styling elements which helps in styling website elements very easy. The above experiment has been styled using various bootstrap elements like badge,glyphicons,grid styling,form-groups,progress-bar and img-responsive.
Bootstrap class badge is used to indicate the numberical indicator of the number of items. In the above case, 4 & 2 indicate the number of folders and disk drives.

Code Snippet:

<div class="col-sm-2">
<span class="glyphicon glyphicon-play"></span>
<label>Folders</label>
<span class="badge">4</span>
</div>

Glyphicon is a small icon which can symbolize things information like print,arrows,search,download etc. In this case the glyphicon 'play' is used to appear before the group header name.

Code Snippet:

<div class="col-sm-3">
<span class="glyphicon glyphicon-play"></span>
<label>Devices and Drives</label>
<span class="badge">2</span>
</div>

A progress bar can be used to show a user how far along he/she is in a process. In this experiment the progressbar has been used to show the status of the disk space.

Code Snippet:

<div>
<div class="progress-bar progress-bar-warning" role="progressbar" style="width:60%">
Used space
</div>
<div class="progress-bar progress-bar-success" role="progressbar" style="width:40%">
Free Space
</div>
</div>


Grid system is used to align items based on 12 columns in the page. The colums are responsive and will get arranged automatically when the page is resized. In the code below, the items are arranged in such a way that each of the three items occupy 4 colums adding up to 12.

Code Snippet:

<div class="form-group row">
<div class="col-sm-4">
<img src="../../images/my_pics/folder_img.jpg" />
<label>Music</label>
</div>
<div class="col-sm-4">
</div>
<div class="col-sm-4">
<img src="../../images/my_pics/folder_img.jpg" />
<label>Pictures</label>
</div>
</div>
