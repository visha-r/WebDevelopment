About the experiment

This experiment uses various bootstrap styling elements like Jumbotron, Pagination, Table & Glyphicons.

A jumbotron indicates a big box for calling extra attention to some special content or information. A jumbotron is displayed as a grey box with rounded corners. It also enlarges the font sizes of the text inside it. Here all the contents have been put inside jumbotron to catch user's attention. The jumbotron itself is placed inside container so it will not exceed the container. If the user wants the jumbotron to occupy the whole page, it can be placed outside the container

Code Snippet:

<div class="container>
<div class="jumbotron">
<div class="row">
<div class="col-sm-4">
</div>
<div class="col-sm-4">
<h3>Customer information</h3>
</div>
<div class="col-sm-4">
</div>
</div>
<div class="row">
<div class="col-sm-10">
</div>
<div class="col-sm-2">
Print this table <span class="glyphicon glyphicon-print"> </span>
</div>
</div>
</div>
</div>

A basic Bootstrap table has a light padding and only horizontal dividers. The table in this experiment uses various styling like table-bordered(table with border), table-striped (table with zebra pattern), table-hover(rows get highlighted when mouse is hovered on them) and table-responsive(table gets resized when based on the page size).

Code Snippet:

<table class="table table-striped table-hover table-bordered table-responsive">

Pagination helps to provide links to pages when we have more than one. It is available in bootstrap as 'pagination' class. The 'active' class says that the current page has be highlighted.

Code Snippet:

<ul class="pagination">
<li class="active"><a href="#">1</a></li>
<li><a href="#">2</a></li>
<li><a href="#">3</a></li>
<li><a href="#">4</a></li>
<li><a href="#">5</a></li>
</ul>
