About the experiment

This experiment portraits jQuery UI elements like datepicker, tabs, autocomplete, slider & dialog.
datepicker : The datepicker as the name implies, can be used to pick a particular date. The date picker can be given different skins depending on the users choice.

Code Snippet:

$("#datepicker").datepicker();

tabs : Tabs are sets of logically grouped content that allow us to quickly flip between them to. Tabs allow us to save space like accordions.

Code Snippet:

$("#tabs").tabs();

autocomplete : Autocompletion is a mechanism frequently used in modern websites to provide the user with a list of suggestions for the beginning of the word he has typed in a text box. jQuery has built-in option for autocomplete feature.

Code Snippet:

var availableTags = [
"ActionScript",
"AppleScript",
"Asp",
"BASIC",
];
$("#tags").autocomplete({
source: availableTags
});


dialog: Dialog is an useful of presenting information to the user. jQuery uses 'dialog' property to make an element as a dialog box.

Code Snippet:

$("#dialog").dialog({
autoOpen: true,
buttons: {
OK: function () { $(this).dialog("close"); }
},
title: "Success"
});


slider: A slider is used whenever a numeric value within a certain range is to be obtained. jQuery's 'slider' property is used to give sliding feature to an html element.

Code Snippet:

$("#slider").slider({
value: 60,
animate: "slow",
orientation: "horizontal"
});
