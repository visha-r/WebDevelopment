jQuery Experiment - Basic jQuery elements

jQuery elements - hide, show, fade, animate, css

About the experiment

jQuery is a feature-rich javascript library that helps to make HTML element manipulation, traversal and event-handling very easy. This experiment portraits basic jQuery events like css,fade, animate, hide & show.
CSS() : The .css() method is a convenient way to get a style HTML elements on the fly. The css method accessed using jQuery can access all properties of normal CSS.

Code Snippet:

$("#blue").click(function () {
$("#myBox").css("background-color", "blue");
});

The code above uses only one CSS property. If multiple CSS properties are to be set, it can be done as follows:

Code Snippet:

$("#blue").click(function () {
$("#myBox").css({"background-color":"blue",
"text-align":"center",
"font-size":10px});
});

fadeTo() : The fadeTo( ) method fades the opacity of all matched elements to a specified opacity and firing an optional callback after completion. Giving opacity less than 1 will fade the item. If the faded item has to be brightened up then the opacity should be made 1. This experiment uses opacity of 0.15 for fading and 1 for brightening.

Code Snippet:

Fading :

 $("#fade").click(function () {
$("#myBox").fadeTo("slow", 0.15);
});

Brightening :

 $("#fade").click(function () {
$("#myBox").fadeTo("slow", 1);
});


hide() & show(): jQuery hide() and show() methods are used to hide and show the element respectively. If a single button has to be used for both hide and show, then toggle() has to be used.

Code Snippet:

hide() :

$("#hide").click(function () {
$("#myBox").hide();
$(".buttonGroup").hide();
});

Show() :

 $("#show").click(function () {
$("#myBox").show();
$(".buttonGroup").show();
});
