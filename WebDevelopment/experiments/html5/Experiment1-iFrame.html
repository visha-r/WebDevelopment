<iframe>

About the experiment

The HTML <iframe> tag is used to specify an inline frame. An inline frame allows you to embed another document within the current HTML document. Inline frames are often used in online advertising, where the contents of the <iframe> is an ad from an external party. 'seamless' attribute of <iframe> speacifies that the frame should look like it is a part of the containing document (no borders or scrollbars). HTML 5 doesn't support attributes such as frameborder, scrolling, marginwidth, and marginheight (which were supported in HTML 4.01). Inline frames are typically created using a second HTML document. This second document contains the content within the inline frame. We should use the iframe src attribute to specify the source of the other document, as well as other attributes to determine the height, width etc. The value of 'src' attribute need not be an url always. It can also be another html document located in the same directory.

Code Snippet:

  <iframe src="Navigation.html" width="600" height="500" seamless></iframe>
