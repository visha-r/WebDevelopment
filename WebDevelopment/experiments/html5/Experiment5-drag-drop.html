About the experiment

Drop and drop concept lets the user to rearrange html elements by means of dragging and dropping. First the element that has to be moved must be selected, dragged over and dropped onto the destination. Drag and drop functionality can be written using javascript or jQuery. There are number of events like dragover,dragleave, dragenter, drag and drop, which gets fired during various stages of the drag and drop operation.

To make an object draggable, its draggable attribute must be set to true and an event listener must be set for onDragStart event.

Code Snippet:

  <div id="boxA" draggable="true" ondragstart="return ondragstart(event)" > <p>Drag Me</p>   </div>

To make an object accept the drop, event listener must be set for its ondragover and ondrop events.

Code Snippet:

  <div id=&boxB& ondrop="drop(event)" ondragover="allowdrop(event)">Drop here</div>

The event listener methods for all the drag and drop events accept Event object which has a readonly attribute called dataTransfer. The dataTransfer object holds data about the drag and drop operation.
