Experiment 3 - HTML5 - date,time & week input types

date,time & week input types

About the experiment

date : The date input type is used to specify date. The date input type also provides a date picker and the way in which the date picker looks depends on the browser that displays it. The format in which the date is displayed depends on the location of the browser. The min and max attributes are used to set the lower and upper bounds for acceptable values. When min or max is specified with some value, the date picker will only enable the dates that are within the given range.

Code Snippet:
        <input type="date" min="2015-01-10" max="2015-01-27">

time :The time input type is used to specify time. Depending on browser support, a time picker can show up in the input field.The min and max attributes are used to set the lower and upper bounds for acceptable values. When min or max is specified with some value, the time picker will only enable the time values that are within the given range.

Code Snippet:
         <input type="date" min="2015-01-10" max="2015-01-27">

week :The week input type is used to select a week and a year. Depending on browser support, a date picker can show up in the input field.The min and max attributes are used to set the lower and upper bounds for acceptable values. When min or max is specified with some value, the date picker will only enable the dates that are within the given range. Step attribute specifies the legal number intervals for an input field. If step is not specified it defaults to 1.

Code Snippet:
        <input type="week" name="week" step=2 min="2015-01-04" max="2015-01-31">
