Experiment 4 - HTML5 - <audio> & <video>

<audio> & <video>


About the experiment

<audio> element is used to embed audio content in html documents. More than one audio source can be specified using the 'src' attribute of <source> element. Currently it only supports audio formats such as MP3,wav and ogg. Any text given between the audio tag will get displayed if the browser doesn't support HTML5 audio. control attribute specifies that the audio controls such as play/pause, mute buttons should be displayed.

Code Snippet:

< audio controls>
<source src="HTML.mp3" type="audio/mp3">
Your browser does not support HTML5 audio
< /audio>


<video> element is used to embed video content in html documents. More than one video source can be specified using the 'src' attribute of <source> element. Currently it only supports audio formats such as MP4,WebM and Ogg. Any text given between the video tag will get displayed if the browser doesn't support HTML5 video. control attribute specifies that the video controls such as play/pause, mute and volume buttons should be displayed.

Code Snippet:
< video width="300" height="225" controls>
<source src="sample_video.mp4" type="video/mp4">
Your browser does not support HTML5 video
< /audio>
