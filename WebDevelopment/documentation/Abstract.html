Brief introduction about the project

The main idea of this project is to create an online shopping website which provides various options to the users like browsing through various types of products, bookmarking their favorite products, adding the required product to the cart, writing reviews for products etc. The website focuses on providing very user friendly experience, so the users can use it with ease. The following are the important features of this online application :
  1. Registering to the application
  2. Login to the application with the registered credentials
  3. Browse through the products based on category
  4. Search products with any valid search keyword like the product name, brand name, manufacturer name etc.
  5. Search for the Best buy stores nearby a particular zipcode
  6. Managing user's own profile
  7. Adding products to the cart and bookmark favorite products
  8. Managing the bookmarks by removing the product which the user no longer wish to have in his list
  9. Managing the cart by updating the product count and removing the products from cart
  10. Write review for a product
  11. Follow/Unfollow a user based on mutual interests
The features included as part of this project have been designed carefully, so the user can have a really good and fairly easy shopping experience!!
