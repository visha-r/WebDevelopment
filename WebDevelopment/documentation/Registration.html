Registration page

The Registration page helps the user to become a member of the application by signing-up. The user can only browse through the products if he is not a member of the application. To experience all the functionalities like writing reviews, bookmarking a product and following other users, the user has to be registered with the application.

Registration page in detail :

When user clicks on the Sign up link, the registration form will look like the following. All the fields have a self-explanatory place holder mentioning what value the form expects. It is clear from the screenshot given below, that the submit button is initially disabled. The submit button will be enabled only if the user provides valid information for all the required fields.

Registration form validation :

Validation has been carried out for all required fields. In addition to checking for blank values, the form also checks for valid username when user tries to register. If the username which the user is trying to choose has already been taken by some other user, error will be thrown. Also error will be thrown if the passwords entered don't match with each other. In short the following are the different errors thrown o this page for invalid data :
  1. First name is left empty
  2. Email is left empty or entered email address is not of the form example@domain.com
  3. Username is empty or username is already taken by someother user
  4. Password is empty or passwords entered don't match each other
