Search Details

When user selects a particular product from the search result by clicking on the product name, product details page will appear. The product details page shows all information about the product like the product features, specifications, reviews and related products.

Search page in detail :

The details page has 3 tabs for Features, Specifications & Reviews and ratings :

Search page options explained :

The details page also has 'Add to cart' button using which the user can add the products to this cart. Related products gets displayed at the right side, using which the user can explore more products in detail. In reviews&ratings section, user can write review for a product which is explained more detail in Reviews documentation
