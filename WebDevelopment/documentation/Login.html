Login page

The Login page helps to authenticate the user using his credentials. Once the user registers to the system, he can use the same username and password that he used while registering. If the fields are left empty while submitting the form, error will be thrown. If the user enters invalid username or password, error will be thrown.

Technologies used :

  • HTML5
  • CSS3
  • AngularJS (validation through directives)
